## Manipulação de Strings em Python

Este trabalho contém uma série de exercícios práticos para aprender a manipular strings em Python.

Cada exercício demonstra diferentes operações, como inverter texto, contar caracteres, verificar palíndromos, converter maiúsculas/minúsculas e verificar anagramas.

## 1. Inverter uma string

Objetivo: reverter a ordem dos caracteres de uma string.

Exemplo:
"Exercicio tpc1" → "1cpt oicicrexE"

Foi utilizado o slicing [::-1] para inverter a string de forma simples e eficiente.

## 2. Contar ocorrências de 'a' e 'A'

Objetivo: contar quantas vezes os caracteres 'a' e 'A' aparecem numa string.

Exemplo:
"Ana Maria Lima" → 'a' aparece 4 vezes e 'A' aparece 1 vez.

Este exercício demonstra a diferença entre letras maiúsculas e minúsculas.

## 3. Contar vogais

Objetivo: contar todas as vogais (a, e, i, o, u), tanto maiúsculas como minúsculas.

Exemplo:
"AAAeieoaiiadiaihsafafakhhasaaiiiiio" → 26 vogais.

Mostra como percorrer a string e verificar cada caractere individualmente.

## 4. Converter para minúsculas

Objetivo: transformar todos os caracteres da string em minúsculas.

Útil para normalizar textos antes de realizar comparações ou contagens.

## 5. Converter para maiúsculas

Objetivo: transformar todos os caracteres da string em maiúsculas.

Semelhante ao exercício anterior, mas aplicado a letras maiúsculas.

## 6. Verificar se é capicua (palíndromo)

Objetivo: verificar se a string é igual à sua inversa.

Exemplo:
"Exercicio tpc1" → False

Uma capicua (ou palíndromo) é uma string que se lê da mesma forma da esquerda para a direita e vice-versa.

## 7. Verificar se duas strings estão balanceadas

Objetivo: verificar se todos os caracteres de uma string estão presentes na outra.

Exemplo:
"abcde" e "edcba" → True

Demonstra como comparar o conteúdo de duas strings sem considerar a ordem dos caracteres.

## 8. Contar ocorrências de uma substring

Objetivo: contar quantas vezes uma substring aparece dentro de outra string.

Exemplo:
"abcdeaeiou" em "aeiouaaabcdeaeiou" → 1 ocorrência.

Útil para análise de texto e identificação de padrões específicos.

## 9. Verificar anagramas

Objetivo: verificar se duas strings são anagramas, ou seja, se contêm os mesmos caracteres em ordem diferente.

Exemplos:

"listen" e "silent" → True

"hello" e "world" → False

Este exercício demonstra como utilizar sorted() para comparar os caracteres independentemente da ordem.
